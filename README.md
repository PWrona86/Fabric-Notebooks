## Overview

This repository contains a collection of individual solutions developed using Microsoft Fabric Notebooks.

Each notebook is designed as a self-contained solution and includes detailed documentation within Markdown cells.

---

## Repository Structure

- Each notebook represents an independent solution.
- No shared framework unless explicitly stated.
- Documentation is embedded directly inside each notebook.

---

## How to Use

1. Clone the repository
2. Upload the notebook to your Fabric workspace
3. Review Markdown documentation
4. Execute cells as instructed

---

## Requirements

- Microsoft Fabric workspace
- Appropriate data access permissions

---

## Documentation Approach

To avoid redundancy, detailed explanations are maintained inside each notebook rather than in this README.
